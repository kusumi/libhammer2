libhammer2 ([v0.2.0](https://github.com/kusumi/libhammer2/releases/tag/v0.2.0))
========

## About

HAMMER2 userspace library in Rust

## Supported platforms

Linux / FreeBSD

## Requirements

Rust 1.81.0 or newer

## Build

    $ make

## Install

    $ make install

## Uninstall

    $ make uninstall
